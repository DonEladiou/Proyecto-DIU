# Proyecto-DIU - BibliotecaUSM

Aplicación web para gestión de biblioteca universitaria con sistema de reservas de salas de estudio.

## Tecnologías

- **Frontend**: React 18 + Vite
- **Backend**: Express.js
- **Estilos**: Tailwind CSS
- **Routing**: React Router DOM

## Instalación

1. Clonar el repositorio y navegar al directorio del proyecto:
```bash
cd Proyecto-DIU
```

2. Instalar dependencias:
```bash
npm install
```

## Desarrollo

Para correr el proyecto en modo desarrollo (con hot-reload):

```bash
npm run dev
```

Esto iniciará:
- **Frontend React**: http://localhost:5173 (Vite dev server)
- **Backend API**: http://localhost:3000 (Express server)

El frontend se conecta automáticamente al backend a través del proxy configurado en Vite.
